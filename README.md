# Nginx-Secure
Nginx-Secure is an attempt to make Nginx as secure as possible by using various modules and other tweaks to the configurations. This is a Docker build that can be used as Web Server or Reverse Proxy.

## Modules Included
- ModSecurity
- IP2Location
